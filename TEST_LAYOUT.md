## Key Layout Features

### Desktop (>768px)
- ✅ Two-column layout: 40% widgets (left) + 60% article (right)
- ✅ Widgets column is sticky (scrolls independently)
- ✅ Article column contains full markdown content with tabs
- ✅ Sidebar does NOT overlap content (proper margin-left)

### Mobile (<768px)
- ✅ Single column stacked layout
- ✅ Widgets stack on top
- ✅ Article content below
- ✅ Bottom navigation visible
- ✅ No horizontal scroll

### Responsive Breakpoint
- **768px** - Switches from mobile to desktop layout
- Uses `@media (min-width: $breakpoint-tablet)`

## Sidebar Fix

**Before (Broken):**
```scss
// ❌ Not working in all browsers
.app-shell:has(.sidebar.collapsed) .main-content {
  margin-left: var(--ai-sidebar-collapsed-width);
}
```

**After (Fixed):**
```scss
// ✅ Works everywhere
.app-shell.sidebar-collapsed .main-content {
  @media (min-width: 768px) {
    margin-left: var(--ai-sidebar-collapsed-width);
  }
}
```
